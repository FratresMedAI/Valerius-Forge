# Valerius Forge Changelog

## [Initial Release] - 2026-05-11

### Added
- **Forge Prompt** command — type a brief and forge a production-grade AI agent system prompt or full project spec, streamed token-by-token from your own API key
- **Forge Selection** command — highlight any text in any app and forge it instantly
- Multi-provider BYOK support: OpenAI, Anthropic (Claude), Google (Gemini), xAI (Grok), OpenRouter, Local LLM (Ollama / LM Studio)
- Three forge modes: Auto Detect, Agent Prompt, Full Project
- Client-side content safety gate — blocks harmful requests before any LLM call
- All API keys stored securely in Raycast Preferences (never sent to any server other than your chosen provider)
